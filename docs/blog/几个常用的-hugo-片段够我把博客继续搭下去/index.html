首页文章列表#

有时候只想拿到 blog 目录下的文章：

{{ $posts := where site.RegularPages "Section" "blog" }}
{{ range first 5 $posts }}
  <a href="{{ .RelPermalink }}">{{ .Title }}</a>
{{ end }}

标签链接#

把标签转换成可点击地址：

{{ range .Params.tags }}
  <a href="{{ (printf "/tags/%s/" (. | urlize)) | relURL }}">{{ . }}</a>
{{ end }}

配置里的一个最小菜单#

[[main]]
name = "文章"
url = "/blog/"
weight = 20
